# BuyBook

Aplikasi BuyBook adalah program Java sederhana yang memungkinkan pengguna untuk menghitung total pembelian buku, termasuk diskon dan pajak. Aplikasi ini dirancang untuk memudahkan proses perhitungan biaya saat membeli beberapa buku.

## Deskripsi

BuyBook memberikan pengguna kemampuan untuk:

- Menginput informasi pembeli
- Menambahkan beberapa buku dengan harga yang ditentukan
- Menghitung total biaya setelah menerapkan diskon dan pajak
- Menampilkan kwitansi pembelian yang jelas dan informatif

## Fitur

- **Input Nama Pembeli**: Memungkinkan pengguna untuk memasukkan nama mereka.
- **Menambahkan Buku**: Pengguna dapat menambahkan beberapa buku dengan judul dan harga. Aplikasi akan terus meminta input hingga pengguna memutuskan untuk berhenti.
- **Perhitungan Diskon**: Menghitung jumlah diskon berdasarkan persentase yang dimasukkan oleh pengguna.
- **Perhitungan Pajak**: Menghitung pajak berdasarkan subtotal pembelian.
- **Kwitansi Pembelian**: Menampilkan rincian pembelian, termasuk subtotal, diskon, pajak, total biaya, dan biaya akhir.

## Prerequisites

Sebelum menjalankan aplikasi ini, pastikan Anda telah menginstal:

- Java Development Kit (JDK) versi 8 atau yang lebih baru
